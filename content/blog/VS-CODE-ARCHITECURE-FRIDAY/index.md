5. Short CUTS
    ### Windows
    1. Ctrl + f -> to Find 
    2. Esc to close find bar
    3. Ctrl + Shift + [ -> Code Folding
    4. Ctrl + Shift + ] -> Code Expanding
    5. Ctrl + D => To select a word
    6. Alt  + UP => move up
    7. Alt + Shift + UP => Duplicate

### MAC
    1. Cmd + f -> to Find 
    2. Esc to close find bar
    3. Cmd + alt + [ -> Code Folding
    4. Cmd + alt + ] -> Code Expanding
